# Sample

## UDP

## One-way communication

```txt
    Client                                   Server
   ┌───────────────────┐                    ┌───────────────────┐
   │ socket()          │                    │ socket()          │
   │                   │                    │                   │
   │ Create a socket   │                    │ Create a socket   │
   └─────────┬─────────┘                    └─────────┬─────────┘
             │                                        │
   ┌─────────▼─────────┐                              │
   │ bind()            │                              │
   │                   │                              │
   │ Connect           │                              │
   └─────────┬─────────┘                              │
             │                                        │
   ┌─────────▼─────────┐                    ┌─────────▼─────────┐
   │ sendto()          │ data               │ recv()            │
   │                   ├────────────────────►                   │
   │ Send data         │                    │ Receive data      │
   └─────────┬─────────┘                    └─────────┬─────────┘
             │                                        │
   ┌─────────▼─────────┐                    ┌─────────▼─────────┐
   │ close()           │                    │ close()           │
   │                   │                    │                   │
   │ Close a socket    │                    │ Close a socket    │
   └───────────────────┘                    └───────────────────┘

```

## Two-way communication

```txt
    Client                                   Server
   ┌───────────────────┐                    ┌───────────────────┐
   │ socket()          │                    │ socket()          │
   │                   │                    │                   │
   │ Create a socket   │                    │ Create a socket   │
   └─────────┬─────────┘                    └─────────┬─────────┘
             │                                        │
   ┌─────────▼─────────┐                              │
   │ bind()            │                              │
   │                   │                              │
   │ Connect           │                              │
   └─────────┬─────────┘                              │
             │                                        │
   ┌─────────▼─────────┐                    ┌─────────▼─────────┐
   │ sendto()          │ data request       │ recvfrom()        │
   │                   ├────────────────────►                   │
   │ Send data         │                    │ Receive data      │
   └────┬─────────▲────┘                    └────┬─────────▲────┘
        │         │                              │         │
   ┌────▼─────────┴────┐                    ┌────▼─────────┴────┐
   │ recv()            │ data reply         │ sendto()          │
   │                   ◄────────────────────│                   │
   │ Receive data      │                    │ Send data         │
   └─────────┬─────────┘                    └─────────┬─────────┘
             │                                        │
   ┌─────────▼─────────┐                    ┌─────────▼─────────┐
   │ close()           │                    │ close()           │
   │                   │                    │                   │
   │ Close a socket    │                    │ Close a socket    │
   └───────────────────┘                    └───────────────────┘
```
